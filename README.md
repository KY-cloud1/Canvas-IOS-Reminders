# UCI Canvas Assignment Fetcher

![Python](https://img.shields.io/badge/Python-3.13.7-blue)
![FastAPI](https://img.shields.io/badge/FastAPI-0.128.0-lightgrey)

## About

A Python module and FastAPI server for retrieving upcoming assignments from the UCI Canvas LMS using the Canvas GraphQL API.
Includes documentation on how to use the local host output with the iOS Shortcuts app to import upcomming assignments into reminders
with the iOS Reminders app.

---

## Features

- Fetch assignments for all courses along with their course, name, and due date.
- Filter assignments due within a specified number of weeks.
- Easy to adapt for other universities by updating the base Canvas url.
- Server module that runs FastAPI server for the user without command line usage.
